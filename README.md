# StoreKitFlow

SwiftUI-first StoreKit 2 framework that simplifies in-app purchases with a protocol-based service layer, reactive store, and structured transaction logging — configured in one `StoreKitFlowConfiguration` call. Includes a full interactive API explorer demo app.
